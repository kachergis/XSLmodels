# Run XSL model

Run XSL model

## Usage

``` r
xsl_run(model, data, control = xslControl())
```

## Arguments

- model:

  An object of class xslMod.

- data:

  An object (or list of objects) of class xslData.

- control:

  Control arguments returned by `xsl_control()`.

## Value

List of fits, sse, unweighted_sse
